# webpack-react-todolist

For learning. Imitate [dida365.com](https://dida365.com).

# The latest features

You can edit by clicking a list item.

# Preview Online

[https://realduolaf.github.io/demo/myy-todo-list/](https://realduolaf.github.io/demo/myy-todo-list/)

# Installing & Building

```
yarn
yarn build
```

or use npm

```
npm install
npm run build
```

# Debugging

```
yarn start
```

or use npm

```
npm start
```

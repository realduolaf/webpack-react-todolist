# webpack-react-todolist

模仿[滴答清单 Web 版](https://dida365.com)。用于学习。

# 最新特性

目前可以通过点击清单项文字的方式直接编辑清单

# 在线预览

[https://realduolaf.github.io/demo/myy-todo-list/](https://realduolaf.github.io/demo/myy-todo-list/)

# 安装 & 构建

```
yarn
yarn build
```

或者，如果使用 npm

```
npm install
npm run build
```

# 本地调试

```
yarn start
```

或者，使用 npm

```
npm start
```
